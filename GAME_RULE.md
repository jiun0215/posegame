# 게임 규칙: Fruit Catcher 🍎

> 하늘에서 떨어지는 과일을 받아 점수를 획득하는 게임입니다!

## 🎮 게임 제목
**Fruit Catcher (과일 받기 게임)**

## 📝 게임 설명
화면 위쪽에서 맛있는 과일들과 피해야 할 폭탄이 떨어집니다.
플레이어는 바구니를 좌우로 움직여 과일을 받고, 폭탄은 피해야 합니다.

## 🎯 게임 목표
- 제한 시간(60초) 동안 최대한 많은 과일을 받아 최고 점수를 기록하세요.
- 폭탄을 받으면 점수가 깎이거나 게임이 종료될 수 있으니 주의하세요!

## 🕹️ 게임 규칙

### 기본 규칙
1. **시작**: "Start" 버튼을 누르면 게임이 시작됩니다.
2. **조작**: 나의 몸(포즈)을 움직여 화면 속 바구니를 조작합니다.
   - **왼쪽 기울이기**: 바구니를 왼쪽으로 이동
   - **오른쪽 기울이기**: 바구니를 오른쪽으로 이동
   - **정면(차렷)**: 바구니를 중앙에 위치
3. **점수**: 과일을 바구니로 받으면 점수 획득!
4. **종료**: 시간이 종료되거나, 폭탄을 받으면 게임 오버.

### 아이템 종류
| 아이템 | 효과 | 점수 |
|---|---|---|
| 🍎 사과 | 기본 과일 | +100점 |
| 🍌 바나나 | 맛있는 과일 | +200점 |
| 🍓 딸기 | 희귀한 과일 | +300점 |
| 🎁 랜덤박스 | 무엇이 들었을까? | -200 ~ +600점 (랜덤) |
| 💣 폭탄 | 받으면 안돼요! | -500점 (또는 게임오버) |

## 🎨 필요한 포즈 목록 (Teachable Machine 학습용)

| 포즈 이름 (Label) | 설명 | 게임 동작 |
|---|---|---|
| **Left** | 몸을 왼쪽으로 기울임 | 바구니를 왼쪽(Left) 구역으로 이동 |
| **Center** | 정면을 보고 서있음 | 바구니를 중앙(Center) 구역으로 이동 |
| **Right** | 몸을 오른쪽으로 기울임 | 바구니를 오른쪽(Right) 구역으로 이동 |

> *팁: 모델 학습 시 `Left`, `Center`, `Right` 대소문자를 정확히 지켜주세요!*

## 📊 점수 및 레벨 체계
- **레벨업**: 점수가 **3000점** 쌓일 때마다 레벨이 오릅니다!
  - 레벨업 시 게임이 일시 정지되고, 3가지 아이템 중 하나를 선택할 수 있습니다.
- **아이템(능력) 선택**:
  1. 🛡️ **방어막**: 폭탄을 1회 막아줍니다. (중첩 가능)
  2. 🍀 **행운**: 폭탄이 나올 확률이 줄어듭니다.
  3. 💰 **탐욕**: 랜덤 박스의 최소/최대 점수가 증가합니다.
- **콤보**: 폭탄 없이 연속으로 과일을 5개 받으면 보너스 점수(+500점)!

## ⏱️ 시간 설정
- **제한 시간**: 60초
- **아이템 생성 속도**: 초기엔 1.5초마다 1개 생성

## 💡 개발 팁
- `gameEngine.js`에서 아이템이 떨어지는 물리 로직(y좌표 증가)을 구현해야 합니다.
- 바구니의 위치는 `poseEngine.js`에서 인식된 포즈(`Left`, `Center`, `Right`)에 따라 즉시 변경됩니다.
